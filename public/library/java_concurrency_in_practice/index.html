How to use this library

workflow

Read the mental model card, then open the lab and break it. Most concurrency bugs are “works until it doesn’t”. Your goal is to feel the failure mode.

Apply the production rules to your own code: identify shared state, decide a policy, document it, and test under contention.

What you get (and what you don’t)

copyright-safe

This zip does not include the book’s copyrighted text. Instead it provides original explanations, simulations, and example code inspired by the chapter topics.

Use it as a study companion and a sandbox to build intuition.
