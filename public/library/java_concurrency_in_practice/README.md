# JCIP Interactive Atlas (Offline HTML)

This is a chapter-by-chapter interactive learning library inspired by *Java Concurrency in Practice*.

## How to use
1. Unzip the folder.
2. Open `index.html` in a browser (Chrome/Edge/Firefox).
3. Press `/` to search anywhere; `Esc` clears search.

## What’s inside
- `index.html`: chapter navigator + entry points
- `chapter-XX-*.html`: one premium page per chapter with labs + deep notes
- `cheatsheet.html`, `glossary.html`, `patterns.html`: reference pages
- `assets/styles.css`: design system (no external fonts/libs)
- `assets/app.js`: interactions + labs (Canvas-based; no dependencies)
