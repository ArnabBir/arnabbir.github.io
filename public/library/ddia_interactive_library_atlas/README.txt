DDIA Atlas — Interactive Chapter Library (Offline)

How to use:
1) Unzip the folder.
2) Open index.html in any modern browser (Chrome/Edge/Firefox/Safari).
3) Use Ctrl/⌘+K to search chapters.
4) Toggle Dark/Light mode in the top right.

Notes:
- This is an original learning aid and does not reproduce the book text.
- Labs are simplified intuition pumps, not production sizing tools.

Folder layout:
- index.html
- chapters/ (one page per chapter)
- assets/ (CSS + JavaScript)
